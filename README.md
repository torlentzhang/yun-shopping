# 云逛街商城

## 一、项目启动

### 安装项目依赖的所有第三方库（根据package.json中的配置）安装依赖

```
npm install
yarn install
```

### 启动本地开发服务器运行

```
npm run serve
yarn serve
```

### 构建生产环境优化的代码（用于部署）

```
npm run build
yarn build
```

### 检查代码

```
npm run lint
```

### 自定义配置

See [Configuration Reference](https://cli.vuejs.org/config/).（Vue-cli配置参考）
